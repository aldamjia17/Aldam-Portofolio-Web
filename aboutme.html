Tentang Saya

Saya lahir pada 19 Maret 2008 di Bone, Sulawesi Selatan. Waktu kecil saya senang menonton film pertarungan, dan bermimpi bahwa saya ingin menjadi astronom, karena saya sangat suka tentang sains teknologi

Di tingkat SMP saya aktif mengikuti ekstrakurikuler pencak silat Tapak Suci dan Rohis. Di ekstrakurikuler Pencak silat saya pernah menjabat sebagai wakil ketua dan Rohis saya menjabat sebagai ketua. pada saat saya masih SMP juga saya pernah mengikuti kelas coding membuat aplikasi. Saat SMA saya memilih jurusan IPA dan saya memasuki ekstrakurikuler pandu digital — di sana saya mulai membuat proyek sederhana seperti poster digital. Di mana jurusan dan ekstrakurikuler ini mendukung minat saya dalam bidang sains dan teknologi.

Saat ini saya terus mengembangkan keterampilan di bidang pemrograman web (HTML, CSS, JavaScript) dan desain UI. Saya percaya pembelajaran berkelanjutan dan praktik langsung adalah kunci untuk menjadi pengembang yang terampil dan kreatif. Selain itu saya juga aktif berpartisipasi dalam komunitas belajar online dan proyek kolaboratif.

Di luar sains dan teknologi, saya juga mempelajari bahasa Jepang.
